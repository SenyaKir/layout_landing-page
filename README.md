# Landing Page

Сторінка компанії BOSE — це адаптивний веб-інтерфейс, що містить продукцію компанії, категорії товарів, інформацію про бренд та форму для зворотного зв’язку.
Проєкт реалізовано з використанням методології BEM та препроцесора SCSS, що забезпечує чисту, зрозумілу та масштабовану структуру стилів.
Для побудови макета застосовано:
Flexbox — для створення адаптивного та зручного в користуванні хедера;
CSS Grid — для компонування основного контенту та каталогу товарів.
Сторінка містить анімовані елементи та декоративні ефекти, які покращують візуальне сприйняття й підвищують інтерактивність інтерфейсу.
